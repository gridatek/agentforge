# AgentForge Console

Minimal Angular 18 (standalone) admin console for AgentForge: chat, inline
citations, redaction notices, and the human-in-the-loop **approval queue**.

```bash
npm install
npm start        # → http://localhost:4200 (expects the API on :8000)
```

This is an intentionally small scaffold — Phase 5 of the roadmap. Live trace
inspection and eval-score dashboards are the natural next additions.
